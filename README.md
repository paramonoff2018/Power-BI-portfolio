# Power BI Dashboard – E‑Commerce Sales Analytics

**Интерактивный дашборд** для анализа продаж интернет-магазина. Реализован в **Power BI Desktop** с тёмной темой и яркими неоновыми акцентами (стиль «Krupp»). Данные содержат информацию о клиентах, товарах и транзакциях за 2023–2024 гг.

## 📌 Ключевые метрики (меры)
- **Общая выручка** (`SUM(Transactions[TotalValue])`)
- **Количество заказов** (`COUNTROWS(Transactions)`)
- **Средний чек** (`DIVIDE([Общая выручка], [Количество заказов])`)

## 📊 Визуализации на дашборде
- **Карточки KPI** – быстрый взгляд на ключевые показатели.
- **Линейный график** – динамика выручки по месяцам.
- **Столбчатые диаграммы** – выручка по детальным регионам (South America, North America, Asia, Europe) и по макрорегионам (America vs Other).
- **Таблица топ‑10 товаров** – рейтинг продуктов по выручке и количеству продаж.
- **Срезы (фильтры)** – по региону, макрорегиону и категории товара.

## 🛠 Особенности обработки данных
- Устранены дублирующиеся столбцы при импорте CSV.
- Созданы дополнительные поля:  
  - `DetailedRegion` и `MacroRegion` (группировка Америки)  
  - Полное имя клиента (`CustomerName`) из двух частей.
- Построены связи между таблицами (`Customers` – `Transactions` – `Products`).

## 🎨 Оформление
- Тёмный фон страницы (`#1A1A1A`).
- Яркие цвета для графиков.
- Тень и полупрозрачный фон визуалов для «реалистичного» объёма.

## 📁 Как использовать
1. Скачайте файл `Amazon_Style_Dashboard.pbix`.
2. Откройте в **Power BI Desktop** (бесплатная версия).
3. Используйте срезы для фильтрации данных по регионам и категориям.
